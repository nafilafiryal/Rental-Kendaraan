# Vehicle Rental Information System

The system developed in this assignment is the **Vehicle Rental Information System**, which is a database system designed to help manage operational activities in a rental business. This includes vehicle inventory management, customer registration, rental transactions (with driver options), vehicle returns with automatic fine calculation, and comprehensive analytical reporting using **Advanced Database** features like Materialized Views and Stored Procedures.
